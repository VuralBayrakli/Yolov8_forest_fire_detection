# YOLOv8 ile Yangın Tespiti

Uzun dönem stajyerlik yaptığım Canovate Ballistic şirketinde yer aldığım Yangın Tespiti projesinde YOLOv8 modelini kendi oluşturduğumuz ateş ve duman içeren veri seti ile eğiterek yüksek doğrulukta ve az hata ile yangın tespitini gerçekleştiren bir model geliştirmeyi amaçladık.
